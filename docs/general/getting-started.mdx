# Decoo 是什么

Decoo 是 DCF (Decentralized Cloud Foundation) 基金会资助的 Web 3.0 服务提供商，旨在提供基于IPFS的分布式存储服务。Decoo 的愿景是为遍布全球的用户创建简单易用的 Web 3.0 服务基础设施 - 一个真正去中心化的分布式云存储。

# Decoo 如何为您提供服务

Decoo 基于开放的 [IPFS](https://ipfs.io/) 协议提供开发者友好的 HTTP API 以实现文件的分布式存储与访问，致力于打造共建共享的分布式存储云。
